# Firmware

This firmware is for a custom 12-key macropad using a Seeed Studio XIAO RP2040.

## Features
- 12-key switch matrix
- EC11 rotary encoder
- 0.91 inch OLED connector
- KMK / CircuitPython firmware

## Pin Mapping

Rows:
- ROW0 = D0
- ROW1 = D1
- ROW2 = D2

Columns:
- COL0 = D3
- COL1 = D6
- COL2 = D7
- COL3 = D8

Rotary encoder:
- A = D9
- B = D10
- C = GND

OLED:
- SDA = D4
- SCL = D5
- VCC = 3V3
- GND = GND
